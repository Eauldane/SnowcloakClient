## About

Snowcloak is a Dalamud plugin that allows you to sync your mods with other players so they can see you as you see 
yourself.

## Install

Add the following custom repository to Dalamud: `https://raw.githubusercontent.com/Eauldane/ElfShelf/refs/heads/main/repo.json`
